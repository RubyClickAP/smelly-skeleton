# SmellySkeleton
Silex skeleton, with ansible/vagrant with Nginx/Php5.5/Memcache

## Installation
* `composer create-project theahindle/smelly-skeleton ./`
* `npm install && composer install`
* `vagrant up`
* `open -a "Google Chrome" http://localhost:8888`
